# 0.4.8 — 2026-05-30

## Added

- **`enrichment` block on search and lookup tools** — `totalCount` (true result count before pagination) and optional `notice` (empty-result guidance) now surface in the typed `enrichment` block on all 9 tools: `openfec_search_candidates`, `openfec_search_committees`, `openfec_search_contributions`, `openfec_search_disbursements`, `openfec_search_expenditures`, `openfec_search_filings`, `openfec_search_legal`, `openfec_lookup_elections`, and `openfec_lookup_calendar`. Both `structuredContent` JSON and `content[]` markdown carry the enrichment data.
